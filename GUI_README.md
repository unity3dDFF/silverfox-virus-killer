# 银狐病毒专杀工具 - 图形化界面使用说明

## 启动GUI

### 方法1：使用启动脚本（推荐）

```bash
./start_gui.sh
```

### 方法2：直接运行

```bash
python3 gui.py
```

## 界面说明

### 主界面布局

图形化界面分为以下几个区域：

1. **标题区域**：显示工具名称和版本
2. **控制面板**（左侧）：
   - 扫描按钮：开始扫描系统
   - 清除按钮：清除发现的威胁
   - 修复按钮：修复系统设置
   - 完整处理按钮：执行完整的扫描、清除、修复流程
   - 进度条：显示操作进度
   - 统计信息：显示威胁数、已清除数、已修复数

3. **结果显示区域**（右侧）：
   - 以表格形式显示扫描结果
   - 包含类型、严重性、路径、详情等列

4. **日志区域**（底部）：
   - 显示详细的操作日志
   - 不同类型的信息用不同颜色显示

### 操作流程

#### 1. 扫描系统

1. 点击"🔍 扫描系统"按钮
2. 等待扫描完成
3. 在结果区域查看发现的威胁
4. 在日志区域查看详细信息

#### 2. 清除病毒

1. 扫描完成后，点击"🧹 清除病毒"按钮
2. 等待清除完成
3. 在统计信息中查看已清除的数量

#### 3. 修复系统

1. 清除完成后，点击"🔧 修复系统"按钮
2. 等待修复完成
3. 在统计信息中查看已修复的数量

#### 4. 完整处理

1. 点击"⚡ 完整处理"按钮
2. 系统将自动执行扫描、清除、修复三个阶段
3. 等待所有阶段完成
4. 查看最终结果

### 界面特点

- **实时进度显示**：进度条会实时更新操作进度
- **详细日志**：所有操作都会记录在日志区域
- **结果表格**：扫描结果以表格形式清晰展示
- **统计信息**：实时显示威胁数、已清除数、已修复数
- **颜色标识**：不同级别的信息用不同颜色显示
- **异步操作**：所有操作在后台线程执行，不会阻塞界面

### 注意事项

1. **按钮状态**：操作进行中，按钮会自动禁用，防止重复点击
2. **进度显示**：进度条会显示当前操作的进度
3. **错误处理**：如果操作出错，会弹出错误提示框
4. **完成通知**：操作完成后，会弹出完成提示框

### 跨平台支持

- **Windows**：所有功能完整支持
- **macOS**：所有功能完整支持
- **Linux**：所有功能完整支持

### 故障排除

#### 问题：GUI无法启动

**可能原因**：
- Python版本过低（需要Python 3.8+）
- 缺少tkinter模块
- 缺少psutil模块

**解决方法**：
```bash
# 安装Python 3.8+
# 安装tkinter（通常Python自带）
# 安装psutil
pip3 install psutil
```

#### 问题：界面显示异常

**可能原因**：
- 系统字体缺失
- 分辨率不兼容

**解决方法**：
- 尝试调整窗口大小
- 检查系统字体设置

#### 问题：操作卡顿

**可能原因**：
- 系统资源不足
- 扫描文件过多

**解决方法**：
- 关闭其他程序
- 等待操作完成

### 技术特点

1. **多线程操作**：所有耗时操作在后台线程执行，保持界面响应
2. **实时更新**：界面实时显示操作进度和结果
3. **错误处理**：完善的错误处理和用户提示
4. **跨平台兼容**：使用Python内置的tkinter，无需额外安装GUI库

### 快捷键

目前GUI版本暂不支持快捷键，所有操作通过按钮完成。
